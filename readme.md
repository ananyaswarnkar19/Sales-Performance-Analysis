# 📊 Sales Performance Analysis Dashboard

## 📌 Project Overview

The Sales Performance Analysis Dashboard is an interactive business intelligence application developed using Python, Streamlit, Pandas, and Plotly. The dashboard enables organizations to analyze sales performance, profit trends, customer behavior, product performance, and regional sales insights through dynamic visualizations and data-driven reporting.

The project helps business stakeholders identify revenue opportunities, optimize product strategies, improve profitability, and make informed decisions based on historical sales data.

---

## 🚀 Features

### 📈 Key Performance Indicators (KPIs)

* Total Sales
* Total Profit
* Total Orders
* Total Customers

### 🌎 Regional Sales Analysis

* Sales distribution by region
* Regional performance comparison
* Identification of high-performing markets

### 📦 Category Analysis

* Category-wise sales contribution
* Product category comparison
* Interactive pie chart visualization

### 🏆 Top Products Analysis

* Top 10 best-selling products
* Product revenue contribution
* Product performance tracking

### 👥 Customer Analysis

* Top 10 customers by sales
* Customer spending patterns
* Revenue contribution by customer

### 📈 Monthly Sales Trend

* Monthly sales growth tracking
* Historical sales performance analysis
* Trend identification and forecasting preparation

### 💹 Profit vs Discount Analysis

* Impact of discounts on profitability
* Identification of loss-making transactions
* Business optimization insights

### 🚚 Shipping Mode Analysis

* Sales distribution by shipping mode
* Customer shipping preferences
* Operational performance insights

### 🔍 Interactive Filtering

Users can filter dashboard data by:

* Region
* Category
* Segment

## 📂 Dataset Information

The project uses the Sample Superstore Dataset containing:

| Feature       | Description                |
| ------------- | -------------------------- |
| Order ID      | Unique order identifier    |
| Order Date    | Date of order placement    |
| Ship Date     | Product shipping date      |
| Ship Mode     | Shipping method            |
| Customer ID   | Unique customer identifier |
| Customer Name | Customer name              |
| Segment       | Customer segment           |
| Country       | Customer country           |
| City          | Customer city              |
| State         | Customer state             |
| Region        | Sales region               |
| Category      | Product category           |
| Sub-Category  | Product sub-category       |
| Product Name  | Product name               |
| Sales         | Sales amount               |
| Quantity      | Quantity sold              |
| Discount      | Discount applied           |
| Profit        | Profit earned              |

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ananyaswarnkar19/Sales-Performance-Analysis.git
```

### Navigate to Project Directory

```bash
cd Sales-Performance-Analysis
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
